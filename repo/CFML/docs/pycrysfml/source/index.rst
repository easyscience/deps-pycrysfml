User Guide
==========
PyCrysFML is a Python wrapper of the CrysFML library. CrysFML, which stands for *Crystallographic Fortran Module Library*, is a library written in `Fortran <https://fortran-lang.org/>`_ language that provides a wide range of procedures for fast crystallographic calculations.

This user guide, as well as PyCrysFML, are under development. Last update: 29/10/2024.
