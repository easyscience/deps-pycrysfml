TITLE
=====

Description
-----------

Describes the content of the CFL file.

Syntax
------

.. code-block:: cfl

    TITLE title

---------

.. raw:: html

   <div class="variable-card">

     </div>
   </div>

Examples
--------

  .. code-block:: cfl

   TITLE Crystal structure of LiFePO4.
