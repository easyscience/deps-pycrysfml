Q_COEFF
=======

Description
-----------

.. raw:: html

   Defines the coefficients of the linear combination of basis vectors that define a propagation vector. See keyword <a href="nkvec.html"><span class="keyword">NKVEC</span>.

Syntax
------

.. code-block:: cfl

    Q_COEFF q1 q2 ... qn

Arguments
---------

.. raw:: html

   <div class="variable-card">
     <div class="var-name">q<sub>1</sub>, q<sub>2</sub>, ..., q<sub>n</sub></div>
     <div class="var-type">Type: integer</div>
     <div class="var-desc">
        Coefficients of the linear combination. n is equal to n<sub>q</sub>. See keyword <a href="nkvec.html"><span class="keyword">NKVEC</span>.
     </div>
   </div>

Examples
--------

.. raw:: html

   See keyword <a href="nkvec.html"><span class="keyword">NKVEC</span>.
