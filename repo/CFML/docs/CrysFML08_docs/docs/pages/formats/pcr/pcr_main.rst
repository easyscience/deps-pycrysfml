The PCR Format
==============

A :filext:`.pcr` file is the main input file of the classic FullProf. Our intention is to progressively document all the variables and keywords that can be used in a :filext:`PCR` file throughout the year 2026. Any variables or keywords not documented on this site can be found in the :download:`FullProf_Manual </_static/pdf/FullProf_Manual.pdf>` or in the :filename:`fp2k.inf` file distributed with the classic FullProf Suite.
